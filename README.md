# Smart GPA Calculator

A modern, feature-rich GPA calculator application for Android developed by Paulo Mkenya.

## Features

### Core Features
- **GPA Calculator**: Add multiple courses with custom names, credit hours, and grades
- **Dynamic Course Management**: Add or remove courses instantly
- **Real-time GPA Calculation**: Instant GPA updates as you add/modify courses
- **Comprehensive Results**: Displays total courses, credit hours, GPA, and academic standing

### GPA Scale (5.0 System)
- A = 5.0
- B+ = 4.0
- B = 3.0
- C+ = 2.5
- C = 2.0
- D = 1.0
- F = 0.0

### Academic Standing
- 4.4 - 5.0: Excellent
- 3.5 - 4.39: Very Good
- 2.7 - 3.49: Good
- 2.0 - 2.69: Pass
- Below 2.0: Probation

### Extra Features
- 🌙 Dark Mode / Light Mode toggle
- 💾 Save GPA calculations to history
- 📜 View calculation history
- 🗑️ Delete individual history records
- 🔄 Reset all data option
- 📱 Responsive Material 3 design
- ✨ Smooth animations

## Technology Stack
- Flutter (Latest Stable)
- SharedPreferences for local storage
- Material 3 Design
- Null Safety enabled
